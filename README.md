# EteChatBlackcat
End to End Chat Web Application

Clone Project
```
git clone https://github.com/thiensgith/EteChatBlackcat.git
cd EteChatBlackcat/
```

## Required
 [NodeJS] v16 (https://nodejs.org/en/)

## Install and run Server
</br>
In EteChatBlackcat/
</br>

```
cd server/
npm install
npm run dev
//OR
npm run prod
```

## Install and run Client
</br>
In EteChatBlackcat/
</br>

```
cd client/
npm install
npm run serve
```

## Browse
(http://localhost:8080)
